Symfony oauth2 bundle based on [league oauth2 server](https://oauth2.thephpleague.com)

# Installation

With orm:

```bash
composer require sonrac/auth-bundle:^orm-1.0
```

Without orm dependency:

```bash
composer require sonrac/auth-bundle:^1.0
```
